Como Jogar

--> Deverá selecionar as cores dos tanques e o bioma em que pretende jogar, de seguida carregue no modo de jogo para começar.

--> Descole o tanque através dos controles disponíveis...

:: Modo 1 Jogador: Cada Bot tem 5 pontos de vida, este mode consiste num mapa onde pode praticar os controles do tanque e a sua pontaria !

:: Modo 2 Jogadores: junte-se com um amigo e tente sair vitorioso da arena !

Controles

Jogador 1

W - Andar para a frente

S- Andar para a trás

A - Virar para a esquerda

D - Virar para a direita

Space - Lançamento de bomba

N - Virar para a esquerda o canhão

M - Virar para a direita o canhão

Jogador 2

Seta para cima - Andar para a frente

Seta para baixo- Andar para a trás

Seta para esquerda - Virar para a esquerda

Seta para direita - Virar para a direita

0- Lançamento de bomba

1 - Virar para a esquerda o canhão

2 - Virar para a direita o canhão
